## Introducción

Visual Studio Code es una de los editores más cónocidos y usados en la actualidad. 
Por defecto ya nos ofrece una gran cantidad de herramientas que podemos usar pero si queremos ir más allá tenemos una gran cantidad 
de plugins para hacerlo más potente.

En este artículo dejaré los principales plugins que uso más en mi día a día y que considero más útiles.

## Plugins

### ESlint
[Descargar plugin](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

Si usas ESlint para arreglar tu código y encontrar errores sin duda este plugin debes instarlo para poder ejecutarlo en VSCode.

### Error Lens
[Descargar plugin](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens)

Junto con el anterior plugin, este es ideal ya que te permite ver los errores inline sin necesidad de estar ejecutando ESlint y viendo los errores en la consola.

### Auto Rename Tag
[Descargar plugin](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)

Con este plugin podrás renombrar las etiquetas html tanto la de apertura como la de cierra cambiandolo solo en una de ellas.

### Tailwind CSS IntelliSense
[Descargar plugin](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)

Este es imprescindible en mi desarrollo ya que trabajo bastante con Tailwind y me proporciona todas las sugerencias para los autocompletados.

### JavaScript (ES6) code snippets
[Descargar plugin](https://marketplace.visualstudio.com/items?itemName=xabikos.JavaScriptSnippets)

Sin duda el mejor plugin para el desarrollo con JavaScript ya que te proporciona una gran cantidad de snippets de código con los que podrás desarrollar de una manera más rápida.

### ES7+ React/Redux/React-Native snippets
[Descargar plugin](https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets)

Si trabajas con React y usas VSCode no te puede faltar este plugin. Tendrás snippets de código para todo lo necesario en React, crear componentes, hooks, etc.
